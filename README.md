# 🍁 MapleWorld 楓之世界

楓之谷風格的 2D 橫向捲軸 RPG 網頁遊戲。純 HTML5 Canvas + 原生 JavaScript，零依賴。
所有圖像皆可**程序化繪製**（無素材也能玩），放入對應 PNG 素材會自動升級畫面。

> 🚀 **可玩性擴充中**：五大職業、25 張地圖、20+ 種怪物、4 大 Boss。
> 擴充規劃見 [DESIGN_EXPANSION.md](DESIGN_EXPANSION.md)；美術素材規格見 [assets/ASSET_REQUESTS.md](assets/ASSET_REQUESTS.md)。

## 🎮 怎麼玩

直接用瀏覽器打開 `index.html` 就能玩（建議 Chrome / Edge）。

### 操作

| 按鍵 | 功能 |
|------|------|
| ← → | 移動 |
| Space | 跳躍（↓+Space 下跳穿越平台） |
| ↑ | 爬繩索 / 進入傳送門 / 與 NPC 對話（商店 / 製作 / 任務） |
| X | 普通攻擊（近戰職揮砍 / 遠程職射擊） |
| A S D F G H | 技能快捷列（從技能視窗拖曳技能即可自訂哪個鍵放哪招） |
| Z | 撿取掉落物（按住可連續撿取） |
| 1 / 2 / 3 | 消耗品快捷鍵（可自訂：點 HUD 快捷格 → 點背包消耗品綁定，右鍵清除） |
| I / K / P | 背包 / 技能加點 / 角色資訊 |
| T | 收合 / 展開任務追蹤 |
| M | 音效開關 |
| O / Esc | 開啟設定（音效、重新載入、返回標題切換角色、刪除存檔） |
| 滑鼠 | 點視窗標題列可拖曳；右上角小地圖點擊放大 |

### 遊戲內容

- **5 大職業 + 轉職**：劍士 / 法師 / 弓箭手 / 盜賊 / 海盜，各有專屬屬性與武器；可在村長處 Lv30 二轉、Lv70 三轉，每職業共 8 個技能
- **自訂角色名**：開新角色可輸入名字，存檔選單與 HUD 皆顯示
- **25 張地圖**：草原 → 森林 → 洞窟 → 祭壇 → 楓葉鎮 → 雪原 → 熔岩 → 古城，七大主題世界一線串接
- **20+ 種怪物**：各區小怪 + 4 大 Boss（蘑菇王、雪人王、炎魔、黑暗領主，皆有衝撞 / 跳壓震波 / 召喚小怪）
- **40 招技能**：5 職業 × 8 招（一/二/三轉），近戰 / 範圍 / 投射 / 治療 / 增益等類型，各有特效
- **商店系統**：村莊有雜貨 / 藥水 / 裝備 / 寵物商店，買消耗品可選數量、買裝備有確認視窗、清單可捲動
- **裝備系統**：12 個裝備欄位（武器/帽/上衣/下衣/鞋/手套/披風/盾/耳環/戒指/項鍊/腰帶）、多階裝備、職業限定武器
- **材料系統**：怪物掉落製作原料（製作配方規劃中）
- **寵物系統**：一開始就有一隻寵物跟隨，會隨時間自動幫忙打怪
- **背包分頁**：全部 / 消耗 / 裝備 / 材料 分頁；可向商人花楓幣擴充格數
- **商店 NPC**：每張地圖都有雜貨商人，可買藥水卷軸、賣道具（可選數量）、擴充背包
- **村莊與 NPC**：楓葉鎮有村長、獵人、草藥師、鐵匠，各有小故事與任務
- **任務系統**：接任務 → 打指定怪物 / 收集指定素材 → 回村交付領楓幣、經驗、裝備（HUD 即時追蹤）
- **製作系統**：鐵匠用材料 + 楓幣打造藥水與裝備（8 種配方）
- **背景音樂**：依主題程序合成 BGM（也支援放入真實音檔），M 鍵開關
- **隨機裝備數值**：每件掉落/製作/任務/購買的裝備數值都不同，分普通/優良/稀有/傳說四稀有度（外框顏色標示），背包移到裝備上會顯示與目前裝備的數值差距（▲ 變強 / ▼ 變弱）
- **小地圖**：右上角顯示地形、NPC、傳送門與自己的位置，點擊可放大（含怪物分布）
- **多角色存檔**：3 個存檔欄位，可養成不同職業、各自獨立進度；全程自動存檔，設定頁可返回標題切換角色或刪除存檔
- **可拖曳視窗 + 自訂快捷鍵 + 設定頁**：視窗可拖曳；消耗品快捷鍵 1/2/3 可自訂；設定頁含音效、重新載入最新版（避免快取）、返回標題、刪除存檔（皆有確認）
- **自動存檔**：進度存在瀏覽器 localStorage，關掉再開繼續玩

## 💰 遊戲經濟設計

整體經濟以「**打怪→變強→挑戰更強的怪**」為核心循環，楓幣（meso）是唯一貨幣，
材料是製作與任務的中介資源。設計目標：消耗品用買的、裝備靠打/做/任務拿、楓幣永遠略嫌不夠用。

### 楓幣循環
| 來源（Faucet） | 去處（Sink） |
|------|------|
| 怪物掉落楓幣（依等級遞增：草原 5–20、洞窟 30–120、雪原 70–320、熔岩 140–560、古城 220–820、Boss 600–20000） | 向商人買藥水 / 卷軸（紅 30 ～ 全恢復 1200） |
| 賣出多餘道具（賣價＝參考價 **60%**） | 製作裝備 / 藥水的工本費（60 ～ 9000） |
| 任務獎勵楓幣（150 ～ 12000） | 擴充背包格（每次 +4 格，費用 **800 × 已擴充次數**，上限 36 格） |

### 裝備取得管道（刻意不在商店賣裝備）
1. **怪物掉落**：各區怪物按機率掉對應階裝備（稀有度越高機率越低，Boss 必掉招牌裝）。
2. **製作（鐵匠）**：用材料 + 楓幣打造，從鐵劍到龍鱗鎧甲共 8 種配方。
3. **任務獎勵**：主線/支線任務送出階段性裝備（巫師帽、鐵盾、楓葉劍、巨龍戒指…）。

### 材料經濟
- 怪物掉落 12 種材料（蝸牛殼、史萊姆膠、水晶碎片、火焰礦石、龍鱗…），掉率約 30–60%。
- 用途：**製作配方**（鐵匠）與**收集任務**（草藥師、鐵匠）。
- 高階材料（龍鱗）僅熔岩/古城高階怪掉落，量少 → 控制頂級裝備節奏。

### 角色成長循環
- 經驗曲線 `expNeed = 18·Lv² + 42·Lv + 40`（二次成長，平滑且可達滿等）。
  - 舊版用 `1.38^Lv` 指數成長，到 Lv30/70 轉職門檻會爆量到天文數字；改為二次曲線後前期快、後期穩。
- 等級 → 解鎖更高階地圖的怪、技能需求等級、裝備需求等級、**轉職**，形成「地圖→裝備→技能→等級」螺旋推進。
- 8 條任務線（村長/獵人/草藥師/鐵匠）串起 4 大 Boss，作為主線里程碑並給大筆 EXP。

### 轉職與技能成長（職業豐富度）
- 每職業 **8 個技能**：一轉 4 招（Lv1 起），二轉 +2 招（Lv30），三轉 +2 招（Lv70）。
- 轉職找**村長 楓伯**：滿等即可轉職，獲得 5 SP 並解鎖該階技能（未轉職的技能在技能視窗顯示為鎖住）。
- 技能威力隨等級線性成長：一轉倍率約 1.0–1.6×、二轉 1.2–1.6×、三轉 2.0–2.6×（對應後期高血量怪）。
- 技能類型：近戰（melee）/ 範圍（aoe）/ 投射（projectile）/ 治療（heal）/ 增益（buff），各有不同特效。
- **技能快捷列**：6 格（A S D F G H），從技能視窗拖曳技能即可自訂哪個鍵放哪招。

### 商店（楓葉鎮三大店家 + 各地雜貨商）
- **雜貨商人**（每張地圖）：消耗品 + 該職業基礎武器；可賣道具、擴充背包格。
- **藥水商店 / 裝備商店 / 寵物商店**（楓葉鎮）：分別專賣藥水、各職武器防具、寵物。
- 買消耗品可**選數量**；買裝備/寵物會跳**確認視窗**；清單可捲動。
- 裝備售價 = 參考價（攻防×8 + HP/MP×0.4 + 需求等級×18 + 40）×1.25；賣出一律 60%。

> 數值集中在 `js/data/`（skills 倍率/消耗、monsters 掉落與經驗、quests 獎勵與配方、items 屬性），方便調整。

## 🛠 技術

- 純前端，無任何框架與建置工具
- Canvas 2D 程序化繪製（角色、怪物、特效、視差背景全部用程式畫）
- WebAudio 合成音效（不用音檔）
- 結構：`js/data/` 遊戲資料（地圖/怪物/道具/技能），`js/entities/` 實體邏輯，其餘為渲染、UI、輸入、主迴圈

## 📁 專案結構

```
index.html
css/style.css
js/
├── config.js      # 遊戲常數與經驗值曲線
├── main.js        # 進入點與主迴圈
├── game.js        # 遊戲狀態機（含選職畫面）、地圖載入、戰鬥結算、存檔
├── data/          # 道具、技能、職業、怪物、地圖資料庫
├── entities/      # 玩家、怪物（含 Boss AI）、掉落物、投射物
├── sprites.js     # 程序化角色繪製
├── render.js      # 視差背景與地圖渲染
├── effects.js     # 傷害數字、打擊特效、公告
├── ui.js          # HUD 與背包/技能/角色視窗
├── camera.js      # 鏡頭跟隨與震動
├── input.js       # 鍵盤滑鼠輸入
├── sound.js       # WebAudio 合成音效
└── utils.js       # 工具函式
```
