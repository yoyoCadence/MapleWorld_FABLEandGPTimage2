# 🍁 MapleWorld 楓之世界

楓之谷風格的 2D 橫向捲軸 RPG 網頁遊戲。純 HTML5 Canvas + 原生 JavaScript，零依賴、零素材檔，所有圖像皆程序化繪製。

## 🎮 怎麼玩

直接用瀏覽器打開 `index.html` 就能玩（建議 Chrome / Edge）。

### 操作

| 按鍵 | 功能 |
|------|------|
| ← → | 移動 |
| Space | 跳躍（↓+Space 下跳穿越平台） |
| ↑ | 爬繩索 / 進入傳送門 |
| X | 普通攻擊 |
| C / V / B / A | 強力斬 / 迴旋斬 / 能量波 / 治癒術 |
| Z | 撿取掉落物 |
| 1 / 2 | 快速喝紅 / 藍藥水 |
| I / K / P | 背包 / 技能加點 / 角色資訊 |
| M | 音效開關 |

### 遊戲內容

- **4 張地圖**：楓葉草原 → 蘑菇森林 → 水晶洞窟 → 王者祭壇
- **6 種怪物**：小藍蝸、綠泡泡、跳跳菇、紫晶菇、石小弟，以及 Boss「蘑菇王」（衝撞、跳壓震波、召喚小怪三種攻擊模式）
- **RPG 系統**：經驗值升級、SP 技能加點（4 個主動技能）、裝備（武器/帽子/衣服/鞋子）、藥水、楓幣
- **自動存檔**：進度存在瀏覽器 localStorage，關掉再開繼續玩

## 🛠 技術

- 純前端，無任何框架與建置工具
- Canvas 2D 程序化繪製（角色、怪物、特效、視差背景全部用程式畫）
- WebAudio 合成音效（不用音檔）
- 結構：`js/data/` 遊戲資料（地圖/怪物/道具/技能），`js/entities/` 實體邏輯，其餘為渲染、UI、輸入、主迴圈

## 📁 專案結構

```
index.html
css/style.css
js/
├── config.js      # 遊戲常數與經驗值曲線
├── main.js        # 進入點與主迴圈
├── game.js        # 遊戲狀態機、地圖載入、戰鬥結算、存檔
├── data/          # 道具、技能、怪物、地圖資料庫
├── entities/      # 玩家、怪物（含 Boss AI）、掉落物、投射物
├── sprites.js     # 程序化角色繪製
├── render.js      # 視差背景與地圖渲染
├── effects.js     # 傷害數字、打擊特效、公告
├── ui.js          # HUD 與背包/技能/角色視窗
├── camera.js      # 鏡頭跟隨與震動
├── input.js       # 鍵盤滑鼠輸入
├── sound.js       # WebAudio 合成音效
└── utils.js       # 工具函式
```
