# 🍁 MapleWorld 楓之世界

楓之谷風格的 2D 橫向捲軸 RPG 網頁遊戲。純 HTML5 Canvas + 原生 JavaScript，零依賴。
所有圖像皆可**程序化繪製**（無素材也能玩），放入對應 PNG 素材會自動升級畫面。

> 🚀 **可玩性擴充中**：五大職業、25 張地圖、20+ 種怪物、4 大 Boss。
> 擴充規劃見 [DESIGN_EXPANSION.md](DESIGN_EXPANSION.md)；美術素材規格見 [assets/ASSET_REQUESTS.md](assets/ASSET_REQUESTS.md)。

## 🎮 怎麼玩

直接用瀏覽器打開 `index.html` 就能玩（建議 Chrome / Edge）。

### 操作

| 按鍵 | 功能 |
|------|------|
| ← → | 移動 |
| Space | 跳躍（↓+Space 下跳穿越平台） |
| ↑ | 爬繩索 / 進入傳送門 / 與 NPC 對話（商店 / 製作 / 任務） |
| X | 普通攻擊（近戰職揮砍 / 遠程職射擊） |
| C / V / B / A | 四個職業技能（依職業不同） |
| Z | 撿取掉落物（按住可連續撿取） |
| 1 / 2 | 快速喝紅 / 藍藥水 |
| I / K / P | 背包 / 技能加點 / 角色資訊 |
| M | 音效開關 |

### 遊戲內容

- **5 大職業**：劍士 / 法師 / 弓箭手 / 盜賊 / 海盜（開新遊戲時選職，各有專屬屬性、武器與 4 個技能）
- **25 張地圖**：草原 → 森林 → 洞窟 → 祭壇 → 楓葉鎮 → 雪原 → 熔岩 → 古城，七大主題世界一線串接
- **20+ 種怪物**：各區小怪 + 4 大 Boss（蘑菇王、雪人王、炎魔、黑暗領主，皆有衝撞 / 跳壓震波 / 召喚小怪）
- **20+ 招式**：近戰 / 範圍 / 投射 / 治療 / 增益（buff）等技能類型
- **裝備系統**：12 個裝備欄位（武器/帽/上衣/下衣/鞋/手套/披風/盾/耳環/戒指/項鍊/腰帶）、多階裝備、職業限定武器
- **材料系統**：怪物掉落製作原料（製作配方規劃中）
- **寵物系統**：一開始就有一隻寵物跟隨，會隨時間自動幫忙打怪
- **背包分頁**：全部 / 消耗 / 裝備 / 材料 分頁；可向商人花楓幣擴充格數
- **商店 NPC**：每張地圖都有雜貨商人，可買藥水卷軸、賣道具（可選數量）、擴充背包
- **村莊與 NPC**：楓葉鎮有村長、獵人、草藥師、鐵匠，各有小故事與任務
- **任務系統**：接任務 → 打指定怪物 / 收集指定素材 → 回村交付領楓幣、經驗、裝備（HUD 即時追蹤）
- **製作系統**：鐵匠用材料 + 楓幣打造藥水與裝備（8 種配方）
- **背景音樂**：依主題程序合成 BGM（也支援放入真實音檔），M 鍵開關
- **自動存檔**：進度存在瀏覽器 localStorage，關掉再開繼續玩

## 💰 遊戲經濟設計

整體經濟以「**打怪→變強→挑戰更強的怪**」為核心循環，楓幣（meso）是唯一貨幣，
材料是製作與任務的中介資源。設計目標：消耗品用買的、裝備靠打/做/任務拿、楓幣永遠略嫌不夠用。

### 楓幣循環
| 來源（Faucet） | 去處（Sink） |
|------|------|
| 怪物掉落楓幣（依等級遞增：草原 5–20、洞窟 30–120、雪原 70–320、熔岩 140–560、古城 220–820、Boss 600–20000） | 向商人買藥水 / 卷軸（紅 30 ～ 全恢復 1200） |
| 賣出多餘道具（賣價＝參考價 **60%**） | 製作裝備 / 藥水的工本費（60 ～ 9000） |
| 任務獎勵楓幣（150 ～ 12000） | 擴充背包格（每次 +4 格，費用 **800 × 已擴充次數**，上限 36 格） |

### 裝備取得管道（刻意不在商店賣裝備）
1. **怪物掉落**：各區怪物按機率掉對應階裝備（稀有度越高機率越低，Boss 必掉招牌裝）。
2. **製作（鐵匠）**：用材料 + 楓幣打造，從鐵劍到龍鱗鎧甲共 8 種配方。
3. **任務獎勵**：主線/支線任務送出階段性裝備（巫師帽、鐵盾、楓葉劍、巨龍戒指…）。

### 材料經濟
- 怪物掉落 12 種材料（蝸牛殼、史萊姆膠、水晶碎片、火焰礦石、龍鱗…），掉率約 30–60%。
- 用途：**製作配方**（鐵匠）與**收集任務**（草藥師、鐵匠）。
- 高階材料（龍鱗）僅熔岩/古城高階怪掉落，量少 → 控制頂級裝備節奏。

### 角色成長循環
- 經驗曲線 `expNeed = 22 × 1.38^(Lv-1)`，上限 Lv.100；任務給大量 EXP 加速前期。
- 等級 → 解鎖更高階地圖的怪、技能需求等級、裝備需求等級，形成「地圖→裝備→等級」的螺旋推進。
- 8 條任務線（村長/獵人/草藥師/鐵匠）串起 4 大 Boss，作為主線里程碑。

> 數值集中在 `js/data/`（items 價格隱含於商店、monsters 掉落、quests 獎勵與配方），方便調整。

## 🛠 技術

- 純前端，無任何框架與建置工具
- Canvas 2D 程序化繪製（角色、怪物、特效、視差背景全部用程式畫）
- WebAudio 合成音效（不用音檔）
- 結構：`js/data/` 遊戲資料（地圖/怪物/道具/技能），`js/entities/` 實體邏輯，其餘為渲染、UI、輸入、主迴圈

## 📁 專案結構

```
index.html
css/style.css
js/
├── config.js      # 遊戲常數與經驗值曲線
├── main.js        # 進入點與主迴圈
├── game.js        # 遊戲狀態機（含選職畫面）、地圖載入、戰鬥結算、存檔
├── data/          # 道具、技能、職業、怪物、地圖資料庫
├── entities/      # 玩家、怪物（含 Boss AI）、掉落物、投射物
├── sprites.js     # 程序化角色繪製
├── render.js      # 視差背景與地圖渲染
├── effects.js     # 傷害數字、打擊特效、公告
├── ui.js          # HUD 與背包/技能/角色視窗
├── camera.js      # 鏡頭跟隨與震動
├── input.js       # 鍵盤滑鼠輸入
├── sound.js       # WebAudio 合成音效
└── utils.js       # 工具函式
```
